# PR Review Assignment Service

Микросервис для автоматического назначения ревьюеров на Pull Request'ы внутри команд.

## Функциональность

- Создание и управление командами и пользователями
- Автоматическое назначение до 2 ревьюеров из команды автора PR
- Переназначение ревьюеров
- Идемпотентный merge PR

## Технологии

| Технология | Назначение |
|------------|------------|
| **Go 1.24.6** | Основной язык разработки |
| **Gin** | HTTP фреймворк для API |
| **GORM** | ORM для работы с PostgreSQL |
| **PostgreSQL** | Основная база данных |
| **Docker** | Контейнеризация приложения |
| **Docker Compose** | Оркестрация контейнеров |
| **golangci-lint** | Статический анализ кода |

## Установка и запуск

### Предварительные требования

- Docker и Docker Compose
- Makefile

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/SANEKNAYMCHIK/Avito-backend-project-autumn-2025.git
cd Avito-backend-project-autumn-2025

# Создание .env файла по примеру
cp .env.example .env

# Запуск всех сервисов
make setup

# Или вручную
make build
make up

# Остановить сервис
make down

# Или сразу остановить и почистить контейнеры и базу данных
make clean

```
